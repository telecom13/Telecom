Dans cette version de l'application, on stocke les données en local avec
localStorage, donc pour faire marcher l'application il vous faut rien à installer.
Pour exécuter l'application, il suffit de lancer le fichier "accueil.html".

Dans le développement de l’application, nous avons utilisé les Framework JavaScript
"jQuery", "Bootstrap"et le Framework "amcharts" pour créer des graphes en 3D.
Nous avons utilisé aussi l’Api Google maps.
La logique métier de l'application, c'est-à-dire toutes les fonctions que nous 
avons créés sont dansle fichier "app.js" (chemin: js/app.js). 
 
NB :
- Nous n’avons pas utilisé le Backbonne, parce que les fonctionnalités
  de l’application ne sont pas très compliquées.  
- Les autres pages HTML  sont dans le dossier "WEB-INF"
